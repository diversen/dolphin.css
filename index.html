dolphin.css

dolphin.css is a drop-in collection of CSS style. It is almost a copy of water.css But with a few modifications.

  1. Simplified. Removed some of the styles.
  2. Split in only three stylesheets. A dark og a light theme, and a main stylesheet.
  3. No build system (is this a feature?). But it means we go down from 996MB to 44KB when developing.

Installation

Go to dolphin.css on github

Goals

  • Responsive
  • Good code quality
  • Good browser support
  • Small size 4KB per file the dist/dir
  • Beautiful
  • No classes

Is it responsive?

Heck yeah! It doesn't include any fancy styles so it's easily mobile responsive. Just add the famous responsive viewport tag and you'll be good to go!

In fact, try resizing this page. Everything flows super nicely as you'll see.
